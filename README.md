## Description
This is a restful api built in python flask.

Features:

- JWT Authenticaton.
- Itercionalization with flask-babel.
- Migrations.
- Flask-Marshmallow.
- Flask-SQLAlchemy.
- Flask-restful.
- Flask-blueprints.
- Cloudinary for images.
- Maigun for emails.
